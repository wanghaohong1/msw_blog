              # 关于作者 - 迷死王

              # 自我介绍

              大家好，我是迷死王。一位热爱记录和分享的开发者，主要关注后端技术、前端应用以及个人效率工具。在学习和工作的过程中，我会把遇到的知识点和实践经验整理成笔记，搭建这个站点的目的就是方便自己复盘、也帮助同路人少踩坑。

              # 目前关注方向

              • Java / Spring 全家桶与常用中间件
              • Vue / React 等前端框架实践
              • 计算机基础复习（408、操作系统、网络等）
              • 自动化部署、个人效率提升
              • 个人项目从设计到上线的全流程记录

              # 网站维护方式

              • 長期整理网络中高质量的公开资料，注明出处后纳入收录
              • 结合自己的学习笔记进行二次整理与补充
              • 通过 issue / PR 的形式欢迎大家纠错或补充内容
              • 保持条理化、工程化的文档风格，方便检索与复用

              # 联系方式

              • GitHub：https://github.com/wanghaohong1 (opens new window)
              • 邮箱：wanghaohong.dev@gmail.com（欢迎交流学习心得）
              • 公众号/社交账号：筹备中，会在站内同步更新

              # 为什么坚持记录

              1. 知识可追溯：重要的学习路线和项目实操步骤都能反复回看，持续优化。
              2. 经验可复用：把踩坑过程写下来，未来遇到类似问题可以快速定位解决方案。
              3. 分享促成长：越能清晰表达，说明理解越透彻，也能收获同行的反馈与建议。

              # 期待与你交流

              如果你也正在学习某个技术栈、准备项目实战或求职面试，希望这里的整理能给你带来启发。也欢迎你通过文末的“完善文章”一起建设这份知识库。
