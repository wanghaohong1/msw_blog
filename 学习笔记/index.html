        # 学习笔记 · 后端技术栈

        聚焦后端工程师核心能力，从整体认知到专项技能，形成可落地的学习路径。

        # 推荐阅读顺序

        1. 后端技术栈导览：梳理宏观全貌与核心能力框架。
        2. Java 与 Spring 生态实战：夯实主流后端语言与框架基础。
        3. 数据库与缓存实践手册：掌握数据持久化、性能优化要点。
        4. 微服务架构与运维实践：构建具备扩展与交付能力的工程体系。

        # 笔记目录

        • 后端技术栈导览
        • [Java 与 Spring 生态实战](./Java 与 Spring 生态实战.md)
        • 数据库与缓存实践手册
        • 微服务架构与运维实践

        # 使用方式

        • 结合自身阶段，针对性选择章节阅读并补充实践项目。
        • 每个章节都包含 “自查清单” 与 “常见问题”，可在复盘时对照完善。
        • 若学习过程中有新的经验或资料，欢迎在站点底部点击“完善文章”提交补充。
