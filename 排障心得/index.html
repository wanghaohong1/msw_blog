        # 排障心得 · 故障复盘库

        记录线上故障处理过程、经验总结与预防措施，打造可复用的排障手册。

        # 推荐阅读顺序

        1. 典型故障案例：按类型了解常见问题的发现、定位与修复思路。
        2. 排障流程模板：复用一套标准化流程，保障排查有序推进。
        3. 工具与命令速览：快速查阅故障排查常用命令与脚本。

        # 目录

        • 典型故障案例
        • 排障流程模板
        • 常用工具命令

        # 使用建议

        • 每次故障处理后更新对应案例，补充触发条件、指标表现和关键时间线。
        • 在流程模板中维护自身团队版本，保证遇到紧急故障时可快速复用。
        • 工具命令可配合脚本库使用，减少重复性操作，提高响应效率。
